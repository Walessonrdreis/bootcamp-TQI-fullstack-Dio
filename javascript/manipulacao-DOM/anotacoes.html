Selecionando os elementos em um página

  • document.getElementById('id_do_elemento');
  • document.getElementsByTagName('nome_da_tag'); Retorna um array, com a tag, para selecionar os itens dentro, basta selecionar com sintaxe do array do javascript
  • document.getElementByClassName('Class_do_elemento'); Semelhante ao tagname, retorna no array que tem a necessidade de ser escolhida pelo índice
  • Com a busca querySelectorALL é possível realizar buscas mais dinâmicas, como nos exemplo a baixo
  • document.querySelectorALL('primeira-classe .segunda-classe')retorna um array;
  • document.querySelectorALL('li .opcao');
